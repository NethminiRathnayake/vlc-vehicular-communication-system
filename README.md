# vlc-vehicular-communication-system
Source Encoded VLC System for  Vehicular Communication using Raspberry Pi Pico RP2040

#  Overview
This project implements a Vehicle-to-Vehicle (V2V) communication system using Visible Light Communication (VLC) with and without source encoding techniques to optimize data transmission.

#  Key Features
- VLC communication using high-power LED  
- 02 Source encoding techniques (Huffman, Lempel-ziv) for efficient data transfer  
- Raspberry Pi Pico-based transmitter and receiver  
- Real-time data transmission and reception  

#  Components and Software Used
- Raspberry Pi Pico RP2040
- 50W 32V LED 
- MicroPython  
- BPW34 Photodiode  
- LM358 Op-Amp
